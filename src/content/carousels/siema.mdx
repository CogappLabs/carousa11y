## Our Implementation Notes

Siema has no built-in pagination. Pagination is optional; if included, use buttons with `aria-label` and `aria-current`.

### Implementation Details

- No built-in pagination; optional to add
- No built-in accessibility features
- Simple API: `new Siema({ selector, loop, onChange })`
- Use `onChange` callback to sync custom pagination state

### Custom Dots Setup

```javascript
const siema = new Siema({
  selector: '.siema',
  loop: true,
  onChange: function() {
    updateDots(this.currentSlide)
  }
})

// Dot click handler
dots.forEach((dot, i) => {
  dot.addEventListener('click', () => {
    siema.goTo(i)
    updateDots(i)
  })
})
```
