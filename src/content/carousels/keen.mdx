## Our Implementation Notes

Keen Slider has no built-in pagination. Pagination is optional; if included, use buttons with `aria-label` and `aria-current`.

### Implementation Details

- No built-in pagination; optional to add
- Use `slideChanged` event to sync pagination state
- Hook-based React API with `useKeenSlider`

### Vanilla JS Setup

```javascript
const slider = new KeenSlider('#slider', {
  loop: true,
  slideChanged: (s) => {
    const index = s.track.details.rel
    updateDots(index)
  }
})

// Navigate to specific slide
slider.moveToIdx(index)
```

### React Setup

```jsx
const [currentSlide, setCurrentSlide] = useState(0)
const [sliderRef, instanceRef] = useKeenSlider({
  loop: true,
  slideChanged: (s) => setCurrentSlide(s.track.details.rel),
})

// Navigate
instanceRef.current?.moveToIdx(index)
```

### Custom Dots

```jsx
{slides.map((_, i) => (
  <button
    key={i}
    onClick={() => instanceRef.current?.moveToIdx(i)}
    aria-label={`Go to slide ${i + 1}`}
    aria-current={i === currentSlide ? 'true' : undefined}
  />
))}
```
